Technology Stack used to build this Website

Front end

  • HTML - The standard markup language for Web pages
  • CSS - The language used to style an HTML document
  • JavaScript - The programming language for the Website
  • Bootstrap - The HTML, CSS, and JavaScript framework for developing responsive, mobile-first website
  • AJAX - The technique for creating fast and dynamic web page - update parts of a web page, without reloading the whole page

Application Layer

  • Python - Programming language for Data Science projects
  • Flask - Web framework. Python module that lets you develop web applications easily
  • Jinja2 - Template engine for Python. A web template system combines a template with a specific data source to render a dynamic web page
